Starry Night

Visit Github
View on Youtube

Role - Design, Coding

Starry Night is a recreation of the painting Starry Night by Vincent Van Gogh using computer graphics with a user interface that allow viewers to interact with the artwork itself. It includes a jigsaw mode and an drawing tool that allows viewer to interact with the original painting.

I started this project inspired by the painting that is famous for the movements implied by the strokes. I felt that animations capture the essence of impressionist artwork, and create an more enriched experience for the viewers than the still image itself.

This project has a time span of less than two months. During that time, I self-learned the 3D graphic module Processing 3D to use with Python3.
