Framework Design:
Financial Data Visualization

Role - API Design, framework core implementation (Java), plugins implementation,
GUI implementation, unit-testing

In this project, I worked in a team to design and implement an extensible visualization and analysis framework in Java. We have chosen our domain to be financial data. Some of the possible extensions we considered were graphs of the historic trends of a stock, â€¨comparisons across different types of stocks, chart of distribution of personal portfolio, etc.

Our framework consists of an interactive GUI tool and underlying interfaces. It supports two types of plugins: data plugins for creating entries for the framework to process data, and display plugins for visualizing data in various forms (e.g pie charts, bar charts, histograms, etc).

I participated in the entire design and implementation process. I was mainly responsible for implementing the core framework, writing sample data and display plugins, and completing test suites.
