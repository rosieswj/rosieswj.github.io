ClearTerms: Web Development

Role - Branding, Design, Web Development, UX / UI, User Testing

ClearTerms is a website developed at the Human-Computer Interaction Institute of Carnegie Mellon. It analyzes the Terms and Conditions of existing websites based on crowdsourced data, and presents the most important statements from those policies to users. In summer 2017 I worked as a front-end developer and UI/UX designer, along with a back-end developer intern. I was mainly responsible for redesigning and implementing the web interface to improve usability and user satisfaction. Some of the major improvements are:

  • Expand the desktop-only interface to a responsive, mobile-friendly platform
  • Uniform the UI design by selecting color palette and creating style sheet
  • Standardize UI components and refactor project structure for easy changes in the future
  • Gamify the crowdsourcing interface encourage more user interactions

This project is still under development. Please message me for more details!
