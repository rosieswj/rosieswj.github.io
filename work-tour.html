CMU Virtual Campus Tour

Visit Github

Role - UX / UI Design, HTML/CSS/Javascript, Responsive Web Design, User Testing

Started in Spring 2017, I work for the Internet of Thing (IOT) Privacy Assistant project at the Institute of Software Research at Carnegie Mellon University. In this project, we use aggregated data to detect user's privacy preferences and help them manage privacy settings.

The IOT assistant has both web and mobile interface. As a graphic designer, I have designed both icons for both UI interfaces. We are currently working on implementing the mobile interface, and I'm looking forward to get to know more of the back-end side of this project.
