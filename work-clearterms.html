Web Development: ClearTerms

Role - Branding, Design, Web Development, UX / UI, User Testing

In summer 2017, I worked as a front-end developer and UI/UX designer on the ClearTerms project at Human-Computer Interaction Institute at Carnegie Mellon University. ClearTerms is a website that analyzes the Terms and Conditions of existing web sites, and presents the most important statements from those policies based on crowdsourced data. Working in a small team with a designer and a back-end developer. We helped improve the existing web interface of ClearTerms project.

I was mainly responsible for redesigning and implementing the UI interface. When I first got my hands on the project, there were a lot of inconsistency and malfunctions on the website, which makes it difficult for users to use. I work closely with my peer to incorporate design principles into the user interface. From this project, I was fortunate to learn about UI/UX design, css/responsible development, and Javascript in depth.
