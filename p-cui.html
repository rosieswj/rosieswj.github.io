Conversational User Interface

View Process book

Role - User Research, Prototyping, Design, Video-editing

Thanks to advancement in machine learning and natural language processing, nowadays conversational user interfaces have becoming more popular among services and products.

In this project, I worked with 3 teammates to build a conversational user interface for the interactions between human and machine in driveless cars. We used patterns of construction such as pre-attention, attention, intent, response/action and error recovery to model our interface.

We first conducted user research with techniques including competitive analysis, comparative matrix, ecosystem collection map, and prototyping session. Then with the refined visual script of pivot scenarios, we made a concept video introducing our concept to potential investors. I participated in the entire process. For the final video, I was mainly responsible for photographing and making all the animation effects.
